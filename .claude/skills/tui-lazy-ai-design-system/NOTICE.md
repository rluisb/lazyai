# Notices

## Brand assets

The wordmark and square mark in `assets/` (`lazyai-wordmark.svg`,
`lazyai-mark.svg`) are **derived** from the CLI's own visual vocabulary
(JetBrains Mono lowercase wordmark, primary/secondary palette). They are
not an official brand mark — the upstream `rluisb/lazyai` repository
ships no logo. Replace these files if and when an official mark is
provided, and update `README.md` accordingly.

## Fonts

The `fonts/` directory ships **JetBrains Mono Nerd Font** (7 weights:
Light, Regular, Italic, Medium, SemiBold, Bold, ExtraBold), licensed
under the SIL Open Font License 1.1. The full license text is in
`fonts/OFL.txt`.

Inter, referenced as a fallback prose face in `colors_and_type.css`,
is loaded from the Google Fonts CDN and is **not** redistributed by
this repository.
