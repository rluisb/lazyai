# Context: Templates

**Category:** Documentation
**Status:** Active

---

## Rule

Use the provided templates for all new documents.

## Rationale

Ensures consistency and completeness across all documentation.

## Guidelines

- **PRD Template:** Use for all new feature requests.
- **Tech Spec Template:** Use for all new technical designs.
- **Tasks Template:** Use for all new task lists.
- **Task Template:** Use for all new individual tasks.
- **ADR Template:** Use for all new architecture decisions.
- **Tech Debt Template:** Use for all new technical debt items.
- **Standard Template:** Use for all new standards.
- **Progress Template:** Use for all new progress reports.

## Enforcement

- PR review checklist
- CI gates for tests and linting
