# Compliance

**Category:** Process
**Status:** Active

---

## Rule

Adhere to all established compliance requirements.

## Rationale

Ensures legal, regulatory, and ethical compliance across the codebase.

## Guidelines

- **Data Privacy:** Protect user data in accordance with GDPR, CCPA, etc.
- **Security:** Follow secure coding practices and conduct regular security reviews.
- **Accessibility:** Ensure the application is accessible to all users.
- **Licensing:** Use open-source software in accordance with license requirements.

## Enforcement

- Compliance reviews
- Security reviews
- Accessibility reviews
- Licensing reviews
