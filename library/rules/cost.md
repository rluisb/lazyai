# Rule: Cost Management

**Category:** Process
**Status:** Active

---

## Rule

Manage cloud costs effectively.

## Rationale

Optimizes resource utilization and reduces unnecessary spending.

## Guidelines

- **Resource Tagging:** Tag all resources for cost allocation.
- **Budget Alerts:** Set up budget alerts for all accounts.
- **Resource Optimization:** Right-size resources and terminate unused ones.
- **Cost Reviews:** Conduct regular cost reviews.

## Enforcement

- Cost management tools
- Budget alerts
- Cost reviews
