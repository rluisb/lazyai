# Prompts Directory

This directory contains prompt templates and examples for AI-assisted workflows.

## Contents
- Prompt templates for common development tasks
- Local examples demonstrating output formats
- Few-shot examples for research, planning, and implementation

## Guidelines
- Each prompt should include a Step 0 (CoT) instruction
- Include at least one few-shot example per prompt
- Add a "Common Mistakes to Avoid" section
- Keep prompts focused on a single task type
