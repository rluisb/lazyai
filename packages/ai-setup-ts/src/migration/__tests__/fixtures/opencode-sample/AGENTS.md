## Project Overview

My Awesome Project is a web application that helps users manage their tasks.

**Stack:**
- Language: TypeScript
- Framework: Next.js
- Database: PostgreSQL
- ORM/Query: Prisma
- Testing: Vitest
- Package manager: npm

## Codebase Map

| Path | Responsibility |
|------|---------------|
| src/app/ | Next.js app router pages |
| src/components/ | React components |
| src/lib/ | Shared utilities |
| src/db/ | Database schema and queries |

## Workflow Rules

- Always write tests for new features
- Use TypeScript strict mode
- Follow the component patterns in src/components/
