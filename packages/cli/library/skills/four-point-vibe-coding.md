# Four-Point Vibe Coding

> Shared skill: point agents to the constitution and clean-code rules.

## When to Use

Use this skill when starting a new task or context window in any CLI agent.

## How to Use

1. Read `canonical/constitution.md` — confirm the four points (WHAT, HOW, DON'T WANT, VALIDATE).
2. Read `canonical/clean-code.md` — apply naming, size, and structure rules.
3. Ask before coding if any of the four points are missing.

## Files

- `canonical/constitution.md` — how we work
- `canonical/clean-code.md` — clean code rules

## Notes

- CLI-agnostic. Works in Claude Code, opencode, and OMP/Pi.
- Keep prompts short; lean on the canonical files for detail.
